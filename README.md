# K-P-Pranav-Portfolio
Personal portfolio website using HTML, CSS, and JavaScript showcasing my projects, skills, internships, and certifications.

## About Me

I am an Electronics and Computer Engineering student specializing in Artificial Intelligence and Machine Learning, with a strong interest in building scalable, real-world solutions using both software and hardware technologies.  
This portfolio serves as a central place to present my technical journey and professional profile.

## Features

- Modern and responsive user interface
- Hero section with social media links (LinkedIn, GitHub, Gmail)
- Skills overview (Software, AI/ML, Electronics & Embedded Systems)
- Internship and professional experience section
- Project showcase with clear descriptions
- Certifications and academic achievements
- Contact form for easy communication
- Optimized for desktop and mobile devices

## Technologies Used

- **HTML5** – Structure and content  
- **CSS3** – Styling, layout, and responsiveness  
- **JavaScript** – Interactivity and dynamic behavior
